1. For Loop

   for (init variable; boolean expression; update ){
            statement ;
   }

   -Step One .. will initialization of variables .
   -Step Two .. check the boolean condition
   -step three.. update or increase loop counter

2. For Each Loop

 - Only when you want loop an array..
 for(type of var: name of array obj){
     statement..
 }


 3. While loop

  if the  boolean expression is true than loop will continue ..

  while (boolean expression ){
        statement.
  }

4. Do While loop

  -- this loop will always execute at least one time..
  -- Body of code first and than check the condition..
  do{

  }while(boolean expression);
